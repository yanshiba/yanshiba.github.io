燕十八开源教程第一波--mysql视频教程!

2012-04-11 | php培训

自2011-12-29日,燕十八发起”PHP开源学习计划“以来,经过4个月的努力, 正式推出mysql视频教程!

对于PHP程序员来说,应该都听过一个单词’LAMP’,即 Linux + Apache + MySQL + PHP,
MySQL 数据库对于PHP程序员的重要性,不必多言.
可以说 —PHPer想拿到4K工资,把PHP的if/else,foreach,smarty等基础知识搞熟就行了,
想拿6K工资,再把常用框架如TP,YII–掌握1-2个就能达到,
但是—如果想拿8K以上工资,没有过硬的MySQL数据库编程知识,是不可能的!

许多刚开始学习PHP的同学,往往不理解MySQL的重要性. 认为–我是来学PHP,学MySQL干吗?
但是,工作半年一年的同学,和老师聊天时最大的感慨就是嫌自己当时没有努力学MySQL!

还有一些同学,因为所在的培训机构刻意淡化MySQL的概念–反正学生也不知道这个重要,淡化了也能忽悠过去.
因此在MySQL的课程安排上,只有区区一天! 每天又只上半天课,折算下来,MySQL只讲了4个小时……

还有一些同学,知道MySQL的重要性,想好好学习MySQL,但可惜目前网络上没有系统,全面又具有深度的MySQL教程,欲学无路!

自学it网公益PHP培训把MySQL的教学上升到前所未有的广度和深度来讲解,前后历时8天,全天讲解,60集重镑推出!
