PHP程序员的学习路线

2012-05-14 | php培训

第一章: PHP程序员的市场
自从2005年web2.0网站兴起以来,以facebook为代表,PHP走向企业级开发.
随着2007年团购网站及电子商务网站爆增,PHP以其”快糙猛”的开发优势,开源平台的特性,
迅速成为WEB开发的首选语言!
“LAMP”组合,如日中天.
2010年,国内团购网站一时达到5000家,对PHP人才的渴求与争夺达到甚至白热化的程度.
PHP,已经是WEB开发领域的王者
不废话,上图上真相:

[caption id=”attachment_453” align=”alignnone” width=”500” caption=”PHP在WEB开发中的份额”][/caption]

通过权威网站w3techs.com的调查,在WEB开发领域,PHP已经占据77.6的份额!
毫无争议的市场第一!

第二章:PHP程序员的技术特点
想学习PHP,必须了解PHP程序员的技术特点.
A:PHP程序员语法简单,学习门槛低,不像c,c++,需要对操作系统的底层机制了解.
B:PHP是名副其实的”WEB语言”,天生为WEB应用,因此PHP程序员99%是做WEB开发,这就要求PHP程序员要会html+css, javascript,MySQL,Ajax,Linux等.
一个C程序员不会mysql,没人说什么,
一个C++程序员不会javascript,也没人说什么,
一个java程序员,不会ajax还是不会有人说什么,
但是,一个PHP程序员如果不会html,js,mysql,ajax,linux那就不是一个合格的PHPer了.
PHP程序员,要求的知识面特点广,是其鲜明特点.
因其”门槛低”的特点,很多原本没有任何编程经验的同学,经过4个月左右的刻苦学习,也能达到初中级PHP程序员的水平,薪资达到5000左右.
可以说:PHP成就了很多人的it梦!

但又因其要求的知识面广,导致一些初学者产生畏惧心理,或者是抓不到重点,乱学一通,走了弯路.

因此,燕老师在此,给大家阐述清楚PHP的学习路线.

第三章:PHP的学习路线

一:html+css+javascript(12天左右)
1.1:html常用标签
1.2:盒模型概念
1.3:div布局+css控制

1.5:javascript操作DOM与CSS
二:PHP基本知识(18天左右)
2.1:变量,常量,函数,表达式,控制结构
2. 3:字符串及字符串函数
2. 4:数组及数组函数

2. 7:文件目录操作
2. 8:cookie&session
2. 9:文件上传下载
2. 10:GD库画图
2. 11:email发送
三:编程思想及模式(8天左右)
3.1:面向对象
3. 2:递归
3. 3:MVC开发模式
四:Mysql知识(4天左右)
4.1:库/表/列的概念
4. 2:建表语句
4. 3:增删改查基本语句
4. 4:左连接,右连接,内连接
4. 5:子查询
4. 6:视图
4. 7:索引
